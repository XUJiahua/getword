# 看拼音写汉字练习纸

一个纯静态 A4 练习纸生成器。打开 `index.html`，粘贴中文字词列表，即可生成适合打印或保存 PDF 的版面。

## 使用

1. 用浏览器打开 `index.html`。
2. 在左侧输入词语，一行一个。
3. 校对拼音，尤其是多音字。
4. 选择练习模式、格子样式、每页词数。
5. 点击“打印 / 保存 PDF”。

## 功能

- 自动标注拼音。
- 支持手动修改拼音。
- 支持加载四年级上册、四年级下册、四年级常见四字词示例词库。
- 支持看拼音写汉字、描红、左范字右抄写、听写纸。
- 支持田字格、米字格、横线。
- A4 纵向打印排版。

完整拼音依赖浏览器从 CDN 加载 `pinyin-pro`。如果离线打开，页面仍可使用，但只内置了示例词的一小部分拼音作为降级。

内置词库是按统编/部编四年级常见字词资料整理的可编辑示例，不等同于官方完整词语表。不同地区、教材版本和学校进度可能不同，打印前建议按老师要求增删。
